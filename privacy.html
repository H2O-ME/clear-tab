清 Tab 浏览器扩展隐私政策

版本 1.1.0 | 最后更新日期：2025年04月03日

前言

《清 Tab 隐私政策》（以下简称“本政策”）适用于 THW（以下简称“我们”）推出的清 Tab 浏览器扩展服务（以下简称“清 Tab”或“本服务”）。在使用本服务之前，请您仔细阅读本政策，以了解我们如何处理您的个人信息。

我们始终承诺保护使用清 Tab 的用户（以下简称“用户”或“您”）的个人信息和隐私安全。本政策旨在向您说明我们在收集和使用您的个人信息时的处理规则，以更好地保障您的权益。

1. 数据收集及使用说明

1.1 本扩展不会收集、存储或传输任何个人身份信息（PII）。

1.2 本扩展仅在本地存储以下匿名数据：

  • 用户设置的搜索引擎偏好
  • API 密钥验证状态（仅本地存储，不进行传输）

1.3 我们使用 Cloudflare 和必应的网站分析工具，这些服务可能会收集您的访问信息，这些信息将作为匿名标识，不会与您的任何个人身份信息关联。

1.4 我们不会收集您的 IP 地址信息，但我们使用的必应网站管理员工具和 Cloudflare 相关服务会用于提供网站流量分析。

Cookie 和类似技术

1.5 我们使用 Cookie 来保存用户偏好。这些 Cookie 仅用于：

  • 记录搜索引擎选择偏好
  • 记录上次搜索历史

1.6 您可以通过浏览器设置管理或禁用 Cookie，但这可能会影响部分功能的正常使用。

2. 数据共享与第三方服务

2.1 在未获得用户许可的情况下，我们不会主动向第三方提供您的个人信息，但以下情况除外：

  • 与国家安全、国防安全直接相关的情况
  • 与公共安全、公共卫生、重大公共利益直接相关的情况
  • 与犯罪侦查、起诉、审判和判决执行等直接相关的情况
  • 出于维护您或他人的生命、财产等重大合法权益的情况

内容审查

2.2 根据相关法律法规和政策，我们禁止以下内容：

  • 违法违禁内容
  • 暴力、恶意软件相关内容
  • 诈骗和侵犯公民隐私的内容

使用的第三方服务

2.3 我们使用以下第三方服务来提升服务质量：

  • Cloudflare（全球 CDN 加速网络，提供网站分析）
  • 必应网站管理员工具（用于匿名数据分析）

2.4 这些第三方服务可能会收集以下匿名信息：

  • 访问时间和频率
  • 设备类型和浏览器信息
  • 国家/地区信息（基于 IP 地址）

3. 数据共享情况

3.1 我们调用了必应、百度、智谱 AI、谷歌和 PearAPI 的相关服务，更多信息请阅读对应服务提供商的相关协议。

4. 数据安全

4.1 我们采用行业标准的安全措施来保护您的信息。

4.2 我们的 API 密钥使用 AES-256 加密算法进行加密处理，以防止盗用。

4.3 我们的程序将尽量在本地运行，但不保证服务提供商的数据安全。

5. 您的权利

5.1 您可以随时通过卸载扩展来删除所有本地存储的数据。

5.2 您可以通过设置搜索引擎来管理您的偏好。

5.3 如果您对本政策有任何疑问或需要删除数据，可以通过以下方式联系我们：thw090221@qq.com

6. 关于浏览器扩展的权限说明

当您启用清 Tab 浏览器扩展时，可能会看到权限提示：“该扩展可以读取和更改您在所有网站上的数据”。我们理解这可能会引起您的担忧，因此希望提供以下说明：

1. 该权限是浏览器扩展提供新标签页替换功能所必需的，使我们能够在您打开新标签页时显示自定义界面。

2. 我们承诺不会收集或传输您的浏览历史、密码或其他敏感信息。

3. 所有数据处理仅在您的本地设备上进行，不会将任何信息发送到我们的服务器。

4. 本程序仅在您的本地设备上进行，不会将任何信息发送到我们的服务器，但相关服务提供商可能需要您的信息以提供相关服务。

7. 运营者基本信息

8. 政策变更

8.1 我们可能会不时更新本政策。更新后的政策将在扩展更新时提供。

8.2 我们会通过扩展内的通知告知您重大变更。

8.3 继续使用扩展即表示您接受更新后的政策。

9. 用户同意

当您使用我们的扩展时，即表示您同意按照本政策所述的方式收集和使用您的数据。
